# Assignment 3 – Application Layer Security

## Overview

This project demonstrates how Java and Spring Boot can be used to improve security at the Application Layer (Layer 7) of the OSI model.

## Features

* User input validation
* Secure password hashing with BCrypt
* SQL injection prevention using prepared statements
* CSRF protection
* Content Security Policy (CSP)
* Role-based access control

## Technologies

* Java
* Spring Boot
* Spring Security
* JdbcTemplate

## Purpose

This project was developed as part of MS-CS 535 to demonstrate common security practices used to protect web applications from attacks such as SQL injection, Cross-Site Scripting (XSS), Cross-Site Request Forgery (CSRF), and unauthorized access.
